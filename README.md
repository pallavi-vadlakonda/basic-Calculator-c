# Basic Calculator in C

This is a simple Basic Calculator program developed using the C programming language. It performs basic arithmetic operations based on user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu-driven program

## How it Works
The program takes two numbers and an operator from the user and performs the selected operation using switch-case logic.

## Technologies Used
- C Programming Language

## How to Run
1. Open the program in any C compiler (like GCC, CodeBlocks, or Turbo C)
2. Compile the code
3. Run the program
4. Enter your choice and input values

## Example
Enter first number: 10  
Enter operator (+, -, *, /): +  
Enter second number: 5  
Result: 15

## Learning Outcome
This project helps in understanding:
- Basic input/output in C
- Conditional statements (switch-case)
- Functions and operators
